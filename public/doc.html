API End Points

  • Get: The GET request, this is to get data from the Prince Georges County dataset
  • Post: this is going to requests the web server accepts the data enclose.
  • Put: we used the put request to retrive search information from the user to render requested info.

Functions

  • renderInfo(res, type): Makes a table version of json data(3 columns) and ammends to to document.
  • getInfo(): The main function that use all the functions above render results based on search bar
  • jsonTable(res): Makes a table version of json data( 2 columns) and ammends to to document.
  • checkComp(res): Goes through the data of test to change background of document to red or green based compliance
  • custom_sort(a, b): Custom sort for year
